# Кой иска да стане физик?

Fullscreen Bulgarian physics quiz game inspired by a TV quiz-show layout.

## Run

Open `index.html` in a browser, or serve the folder locally:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4173/`.

## Audio

The game includes synthesized quiz-show style sounds through Web Audio. If you have licensed audio files, place them in `assets/audio/` and map them in `assets/audio/audio-config.js`.
